Abstract

We introduce Infinigen-Sim, a toolkit which enables users to create diverse and realistic articulated object procedural generators. These tools are composed of high-level utilities for use creating articulated assets in Blender, as well as an export pipeline to integrate the resulting assets into common robotics simulators. We demonstrate our system by creating procedural generators for 5 common articulated object categories. Experiments show that assets sampled from these generators are useful for movable object segmentation, training generalizable reinforcement learning policies, and sim-to-real transfer of imitation learning policies.

Procedurally Generated Simulation Assets

With Infinigen-Sim, we have created procedural generators for doors, toasters, refrigerators, dishwashers, and lamps. We're actively expanding support with the goal of covering all articulated object types. Each generated asset can be exported to simulation-ready formats like URDF, USD, or MJCF using our built-in tools.

Infinigen-Sim gives users full control over asset creation, enabling the design of detailed articulated objects with accurate joint configurations and high visual fidelity. Assets can also leverage Infinigen's procedural material library to introduce rich visual diversity.

Create Your Own Assets

Infinigen-Sim also lets users create custom procedural sim-ready assets for their own tasks. This can be done in three steps.

Step 1

Using Blender Geometry Nodes, a node-based proceudral generation tool, users can build asset generators for their particular task. To introduce articulations, we create custom joint nodes for both revolute and sliding joints.

Step 2

Users can then define their parameter distributions.

Step 3

Finally, using Infinigen-Sim, users can spawn an instance of their assets an export to the simulation format of their choice.
